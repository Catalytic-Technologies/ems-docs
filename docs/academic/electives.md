# Electives

<span className="role-badge role-badge--admin">School Admin</span>

The Electives module manages optional subjects that students choose from a pool of available options. This is commonly used for subjects like Art, Music, Technical Drawing, or languages in upper school.

## How Electives Work

1. The school defines **elective groups** — sets of subjects students must choose from.
2. Each elective group has a defined **number of choices** a student must make (e.g. "choose 2 from Group B").
3. Students (or admin on their behalf) select their electives, and these feed into their timetable and assessments.

## Creating an Elective Group

1. Go to **Settings → Elective Rules** (or **Academic → Electives**).
2. Click **New Elective Group**.
3. Set the group name, minimum and maximum choices per student.
4. Add the available subjects to the group.
5. Click **Save**.

## Assigning Electives to Students

1. Go to **Academic → Electives**.
2. Select the class or student.
3. Click **Assign Electives**.
4. Select the student's choices according to the rules.
5. Click **Save**.

:::tip[Bulk assignment]
Use the bulk assignment tool to process an entire class at once if all students are taking the same elective combination.
:::

## Viewing Elective Enrolments

The Electives list shows:
- Which students are enrolled in each elective subject
- Which students are missing elective assignments
- Enrolment numbers per elective (useful for timetabling room sizes)
