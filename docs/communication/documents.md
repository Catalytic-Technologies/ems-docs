# Documents

<span className="role-badge role-badge--admin">School Admin</span>

The Documents module is a centralised file store for school-wide documents — circulars, policies, calendars, and other files that need to be shared with staff or guardians.

## Uploading a Document

1. Go to **Communication → Documents**.
2. Click **Upload Document**.
3. Select the file (PDF, Word, Excel, or image).
4. Enter a **title** and optional **description**.
5. Choose the **audience**: Staff only, Guardians, or All.
6. Click **Upload**.

Uploaded documents are immediately available to the selected audience.

## Managing Documents

From the documents list you can:
- **Download** any document by clicking it
- **Delete** documents you uploaded (Admin only)
- **Filter** by audience or file type
