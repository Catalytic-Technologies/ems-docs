# Public Admissions Portal

The Admissions Portal is a **public-facing web page** that prospective parents use to apply for their child's admission. No login is required.

## Portal URL

The portal is available at:

```
https://app.ems.school/apply/YOUR_SCHOOL_CODE
```

Your school code is shown in **Settings → School Profile**. Share this link on your website, social media, or printed materials.

## What Applicants See

The portal guides parents through a multi-step application form:

1. **Student details** — name, date of birth, gender, previous school
2. **Guardian details** — primary contact, relationship, phone, email
3. **Supporting documents** — birth certificate, passport photo, previous school report (optional)
4. **Review & submit**

After submission, the parent receives a **confirmation email** with their application reference number.

## Portal Availability

The portal only accepts applications when a cycle is **Active** and within the **Open Date – Close Date** window. Outside this window, the portal displays a "Applications are currently closed" message.

:::tip[Share the direct cycle link]
If you have multiple active cycles (e.g. Grade 1 and Grade 7 simultaneously), applicants are asked to select the appropriate cycle at the start of the form.
:::

## Customising the Portal

Go to **Settings → School Profile** to update:
- School name and logo displayed on the portal
- Welcome message shown to applicants
- Required documents list
