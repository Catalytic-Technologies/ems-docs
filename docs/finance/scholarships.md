# Scholarships

<span className="role-badge role-badge--admin">School Admin</span> <span className="role-badge role-badge--bursar">Bursar</span>

The Scholarships module manages bursaries and fee discounts awarded to students, including sponsor tracking and application management.

## Scholarship Sponsors

Before creating scholarships, set up the sponsors funding them.

1. Go to **Finance → Scholarships → Sponsors**.
2. Click **New Sponsor**.
3. Enter the sponsor's name, contact, and any reference details.
4. Click **Save**.

## Creating a Scholarship

1. Go to **Finance → Scholarships**.
2. Click **New Scholarship**.
3. Fill in:

| Field | Description |
|-------|-------------|
| **Name** | e.g. "Full Bursary 2026", "50% Fee Waiver" |
| **Sponsor** | The funding sponsor |
| **Discount Type** | Percentage or fixed amount |
| **Discount Value** | The discount amount/percentage |
| **Coverage** | Which fee items are covered |
| **Maximum Recipients** | Cap on how many students can hold this scholarship |

4. Click **Save**.

## Awarding a Scholarship

To award a scholarship to a student:

1. Open the scholarship.
2. Click **Add Recipient**.
3. Search and select the student.
4. Set the **start term** and **end term**.
5. Click **Save**.

The scholarship discount is automatically applied to the student's invoices for the covered terms.

## Scholarship Applications

If your school uses an application process for scholarships:

1. Go to **Finance → Scholarships → Applications**.
2. Review submitted applications.
3. Approve or reject each application.
4. Approved applications automatically enrol the student as a recipient.
