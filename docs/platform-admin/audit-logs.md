# Audit Logs

<span className="role-badge role-badge--platform">Super Admin</span>

The Audit Log is a chronological record of all significant actions taken across the platform — who did what, when, and on which record.

## Accessing Audit Logs

Go to **Platform → Audit** to view the full audit trail.

## Filtering Logs

Use the filters to narrow down the audit log:

| Filter | Options |
|--------|---------|
| **School** | Filter to a specific tenant |
| **User** | Filter to a specific user account |
| **Action Type** | Create / Update / Delete / Login / Export |
| **Module** | Students / Finance / Users / Settings / etc. |
| **Date Range** | From and to dates |

## What is Logged

Every significant action is recorded, including:
- User login and logout events
- Student record creates, edits, and deletes
- Fee payments recorded
- Invoices generated
- Report pack generation and downloads
- Settings changes
- User account creation and role changes
- Tenant suspension / reinstatement

## Exporting Audit Logs

Click **Export CSV** to download the filtered audit log. This is useful for compliance reporting or investigating a specific incident.
