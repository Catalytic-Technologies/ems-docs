# Subscriptions & Billing

<span className="role-badge role-badge--platform">Super Admin</span>

The Subscriptions module manages billing plans, school subscriptions, and platform invoices.

## Billing Plans

Define the pricing tiers available to schools:

1. Go to **Platform → Billing → Plans**.
2. Click **New Plan**.
3. Set:
   - Plan name (e.g. "Starter", "Standard", "Enterprise")
   - Monthly/annual price
   - Student capacity limit
   - Features included

## School Subscriptions

Each school has a subscription linked to a billing plan.

1. Go to **Platform → Billing → Subscriptions**.
2. Find the school.
3. View their current plan, next billing date, and payment status.

### Upgrading a Plan
1. Open the school's subscription.
2. Click **Change Plan**.
3. Select the new plan.
4. Confirm — the change takes effect immediately.

## Platform Invoices

Generate and manage invoices sent to schools for their subscriptions:

1. Go to **Platform → Billing → Invoices**.
2. Click **New Invoice** or use bulk invoice generation for all due subscriptions.
3. Send invoices directly from the platform via email.

## Usage Monitoring

Go to **Platform → Usage** to monitor per-school usage metrics:
- Active student count vs plan limit
- Storage used
- API call volume

Schools approaching their limits are flagged for upsell or upgrade.
