# School Calendar

<span className="role-badge role-badge--admin">School Admin</span>

The School Calendar is a shared calendar visible to all staff. Use it to communicate term dates, events, exams, holidays, and closures across the school.

## Viewing the Calendar

Go to **Administration → Calendar** to see the full calendar view. Switch between **Month**, **Week**, and **Day** views using the buttons at the top right.

## Adding an Event

1. Click **New Event** or click directly on a date.
2. Fill in:

| Field | Description |
|-------|-------------|
| **Title** | Event name |
| **Date / Time** | Start and end date/time |
| **All Day** | Toggle for full-day events |
| **Category** | Academic / Sports / Holiday / Meeting / Exam / Other |
| **Description** | Additional details |
| **Audience** | Staff only, or all (including guardians via the app) |

3. Click **Save**.

## Editing or Deleting an Event

Click on an event in the calendar to open it. Use **Edit** or **Delete** as needed.
