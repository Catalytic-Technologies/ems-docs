# Dashboard

The Dashboard is the home screen you see after logging in. It is personalised to your role and shows the most important information at a glance.

## School Admin Dashboard

<span className="role-badge role-badge--admin">School Admin</span>

- Total students enrolled
- Attendance rate today (% of students present)
- Outstanding fee balance across all students
- Recent payments received
- Pending admissions applications
- Quick links to common tasks

## Bursar Dashboard

<span className="role-badge role-badge--bursar">Bursar</span>

- Fee collection summary for the current term
- Total outstanding balance
- Recent payments
- Overdue invoices count
- Income vs budget chart

## Teacher Dashboard

<span className="role-badge role-badge--teacher">Teacher</span>

- Today's classes and periods
- Attendance marked / not yet marked
- Upcoming assessments
- Recent notifications

## Navigating from the Dashboard

Click any metric or chart to drill into the related module for more detail. For example, clicking the outstanding balance figure takes you to the defaulters list.
