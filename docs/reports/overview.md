# Reports & Analytics

<span className="role-badge role-badge--admin">School Admin</span> <span className="role-badge role-badge--bursar">Bursar</span>

EMS provides comprehensive reporting across all modules. Reports can be viewed on screen, exported to CSV, or downloaded as PDFs.

## Reports Centre

Go to **Analytics → Reports** to access the reports centre. Reports are grouped by module:

| Category | Available Reports |
|----------|------------------|
| **Academic** | Attendance summary, attendance by class, assessment results, class performance |
| **Finance** | Fee collection summary, defaulters, payment history, income statement, BvA |
| **Admissions** | Applications by status, cycle summary |
| **Inventory** | Stock levels, movement report, low stock |
| **General** | Student demographics, staff list |

## Running a Report

1. Select a report from the list.
2. Set the **filters** (date range, class, term, etc.).
3. Click **Run Report**.
4. The report loads on screen.
5. Use **Export CSV** or **Download PDF** to save it.

## Report History

Previously generated reports are saved in **Analytics → Report History**. You can re-download them without re-running the filters.

## Dashboards

Each role has a **Dashboard** with live charts and key metrics relevant to their work. Dashboards refresh automatically when data changes.
