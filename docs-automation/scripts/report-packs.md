Welcome to the EMS Report Packs module. This guide shows you how to generate end-of-term report cards for students.

Report packs bring together assessment marks, attendance, and teacher comments into a formatted PDF report card for each student.

To get started, go to Analytics in the left sidebar, then click Report Packs. Click New Report Pack to create a new pack for this term.

Select the term, the classes to include, and the report template you want to use. Then click Create. EMS will prepare the report pack with all the students from the selected classes.

Teachers can now enter their subject marks and comments. Each teacher goes to Analytics, then My Report Packs, opens the pack, and enters marks and a comment for each student in their subject.

The class teacher also adds an overall class teacher comment for each student. Once a teacher has completed their entries, the school administrator can see their progress.

When all entries are complete, the school administrator reviews the pack using the Student Preview. This shows exactly what each student's report card will look like. Any corrections can be made at this stage.

Once you are satisfied, click Generate PDFs. EMS will create a PDF report card for every student in the pack. You can then download a ZIP file containing all the PDFs, or click Send to Guardians to email each parent their child's report card directly.

That is how you generate report packs in EMS.
