Welcome to the EMS Finance module. This guide shows you how to generate fee invoices for students at the start of a new term.

Before generating invoices, make sure your fee structures are set up for the current term. You can find this under Finance, then Fee Structures.

To generate invoices, go to Finance in the left sidebar, then click Invoices. You will see the list of all existing invoices.

Click the Generate Invoices button at the top of the page. A panel will open where you can select the term you want to invoice for, and choose whether to generate invoices for all classes or a specific class.

Click Preview to review a summary of the invoices that will be created. You can see how many students will be invoiced and the total amount. Check this carefully before proceeding.

When you are satisfied, click Generate to create the invoices for all selected students. The system will create individual invoices for each student based on their fee structure.

EMS will not create duplicate invoices. If a student already has an invoice for this term, they will be skipped automatically.

After generation, you can use the Send Notification button to notify guardians by email or SMS that their invoices are ready.

That is how you generate fee invoices in EMS. For more information on recording payments, see the Payments guide.
